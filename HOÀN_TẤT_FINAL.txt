✅✅✅ CẢM ƠN BẠN ĐÃ TÌM RA!

VẤN ĐỀ: Code CŨ addEventListener('play') còn sót lại
         → Tăng count 2 lần (play + ended)
         → 0 + 1 + 1 = 2 (hết luôn!)

ĐÃ SỬA: Xóa addEventListener('play')
        → Chỉ tăng count ở onended
        → 0 + 1 = 1 (đúng!)
